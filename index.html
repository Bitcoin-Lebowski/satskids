Welcome to SatsKids!
👨‍👧‍👦
A Bitcoin savings app for families

SatsKids helps kids learn about Bitcoin, saving, and the value of money, through tasks, savings goals, and weekly sat payments.

Sats are simulated by default. Real lessons, no real money, no real money needed to start.

⚡
How it works
📋Kids tick off their chores during the week
🗓️Saturday morning: parent reviews and approves
⚡Generate an invoice and "pay" with the animation
📈Kids watch their balance grow and learn about Bitcoin price
🔐
Set up parent mode

Parent mode lets you add chores, set sat values, pay invoices, and manage the app. It's protected by a PIN you set yourself.

Kids can view their balance and tick tasks but can't pay themselves or change settings.

💡 Turn on Parent mode on the home screen to get started. You'll set your PIN on first use.
🌐
Want to use real sats?

By default everything is simulated. Great for learning. When you're ready to use real Bitcoin Lightning payments, you can connect to an LNbits backend.

Tap ⚙️ Settings on the home screen to configure a Lightning connection. This is optional. The app works fine without it.

🎓 We recommend starting in simulated mode and switching to real sats once the kids understand how it works.
⚙️ Settings
Lightning Backend (optional)
Connect to a real LNbits instance to use actual Bitcoin Lightning payments instead of simulated sats. Leave blank to keep using simulated mode.
Backup & Restore
Export saves all wallets, balances, tasks and history to a file. Import restores from a previous export.
